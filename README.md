# Wassim Soltani - Portfolio

This is the production build of my personal portfolio website, hosted on GitHub Pages. The site is built using [Astro](https://astro.build/) and deployed automatically via a custom deployment script.

## 🛠️ Technologies Used

- **Astro**: A modern static site generator for fast, optimized websites.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Cloudflare Workers**: A worker for the contact form.

## 📂 Contents

- **`index.html`**: Main entry point.
- **`assets/`**: Static assets (images, fonts, CSS).
- **`skills/`**, **`projects/`**, **`experience/`**, **`contact/`**: Additional pages.
- **Icons & Favicons**: `apple-touch-icon.png`, `web-app-manifest-*.png`, `favicon.*`, `avatar.png`.
- **Sitemaps**: `sitemap-index.xml`, `sitemap-0.xml`.
